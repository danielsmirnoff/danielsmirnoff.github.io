    Lab Rats

    Project Details

    • Game Type: Personal Project
    • Engine: Unity
    • Genre: 3D, Multiplayer, Horror
    • Status: IN DEVELOPMENT

    Overview

    I started this project as a way to learn more about multiplayer development while hoping to eventually release it as a commerical game. The idea was to make a mutliplayer-horror game similar to another indie game: Lethal Company, the current premise is for players to work together to explore an area while completing objectives and collect items; all the while avoiding enemies and hazards.

    Features

    As of right now, I have been the sole developer of the entire project.

    Art

    • Drew up concepts for enemies, and levels.
    • 3D modeled all models, including player model, items, enviornment, enemies.
    • Rigged and animated character models.

    Programming

    • Fully networked item pickup and drop system. Items appear in both UI and on player model for both host and clients. Dropping items removes it from inventory and spawns it at players feet. Items also have functionality that are item specific.
    • Created a networked lobby system using Steam, players can host and join lobbies.
    • Created a fully networked dungeon generator, the only networked part is the seed and the rest is handled by each client. See video above to view the algorithm.
